# Tri2Do
#### It is ONE-STOP Web-App for all trending DSA Sheets. 

## Installation Guide
### clone the project
```
  git clone git@github.com:Ankur-Ydv/Tri2Do.git
```
### go to the project directory
```
  cd Tri2Do
```
### Install dependencies
```
  npm install
```
### Setup Enviroment
```
  1. Create .env file
  2. Provide following keys
  BASE_URL, MONGO_URI
```
### start the app
```
  npm run dev
```
